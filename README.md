## Installation
To install Ngram packages, go to the Playground (Ctrl+OW) in your Pharo image and execute the following Metacello script (select it and press Do-it button or Ctrl+D):
```Smalltalk
Metacello new
  baseline: 'Ngram';
  repository: 'github://olekscode/Ngram/src';
  load.
```
